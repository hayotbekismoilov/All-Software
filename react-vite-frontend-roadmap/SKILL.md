# React Vite Frontend Roadmap

## Purpose
Turn skill lists into a practical roadmap that is production-oriented and hiring-focused.

## Use When
- User asks for a learning roadmap for React + Vite
- User wants practical progression for CRM/dashboard or Telegram WebApp
- User wants weekly execution plan with minimal theory

## Required Stack Defaults
- React + Vite (no Next.js)
- JavaScript (ES6+) or TypeScript from week 5+
- Tailwind CSS
- Zustand
- REST API integration

## Workflow
1. Convert user skill list into phases.
2. Keep practice-to-theory ratio near 80/20.
3. Attach each skill to one mini-task and one real use-case.
4. Build project progression from components to full CRM/dashboard.
5. Add company-style best practices and common mistakes.
6. Output 6-8 week plan with measurable deliverables.

## Phase Contract
- Week 1-2: components + UI system
- Week 3-4: API + state management
- Week 5-6: TypeScript + performance
- Week 7-8: full CRM + production quality

## Output Format
1. Phase roadmap
2. Skill matrix: practical meaning, mini-task, real use-case
3. Weekly plan with 2-3 hour day slices
4. Common mistakes and prevention
5. Hiring-ready portfolio milestones
